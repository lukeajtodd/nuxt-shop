# nuxt-shop - Static Shop

## Stack

- Nuxt
- Sanity.io
- Snipcart
- Tailwind (& Tailblocks)

Static shop built with Nuxt & Sanity. Attempt to build a statically updated shop that houses a small number of items.

## Requirements

- Node@12
- Yarn

## Setup

- `yarn setup`
- Carry out `.env` updates
- `yarn dev`

## Issues

- Sanity ID is hard-coded in `sanity.json`
